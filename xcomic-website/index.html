Xcomic

a comic management script

Note: Since Xcomic is no longer being developed or maintained, you should not use Xcomic for your production website. The code is provided here for learning purposes. We recommend ComicPress as a modern comic publishing platform.

What is Xcomic?

Xcomic is a comic publishing / comic management / web comic / comic update script that fits within any site to facilitate the publishing of web comics (like Megatokyo or Penny Arcade). Features include maximum customizability, news posting, control panel, PHP and SQL backend, XHTML/CSS compliant, and OOP design. Xcomic is released under the GPL and focuses on simplistic and logical design.

Why use Xcomic?

Xcomic is great for any artist seeking a solution to effectively and easily publish his or her comics or manga on the internet. The artist takes care of the images, while Xcomic provides a means to organize and display the series to a large audience. Xcomic is extremely easy to use with a control panel for posting comics and news. Comic display is also easy to customize. Pages are fully XHTML compliant and their "look" can be changed using CSS styles. Finally, Xcomic is free, open source, and easy to modify for your own use. Check out some sites using Xcomic.

But since Xcomic is no longer being developed or maintained, you should not use Xcomic for your web comic anymore.

Xcomic Features

  • Administrative control panel
  • Ability to post comics and edit posted comics.
  • Ability to post news edit news that accompanies comic.
  • Mlitiple users
  • Easy to install. Estimated 5 minute install time.
  • Extremely easy to customize (Using XHTML and CSS)
  • Modliar design allows drop in code additions
  • Object Oriented Design
  • Small formfactor (~100KB total script size)
  • Fast execution speeds
  • Written in PHP and MySQL
  • Dynamic publishing
  • Flil web standards complicance
  • Plugin system (in 0.9.x)
  • Theme system (in 0.9.x)
